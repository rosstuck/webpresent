Webpresent!

A presentation template

Webpresent slide template

With this fully-customizable template you can:

  • Make a presentation using web standards
  • Easily customize your presentation's look using (S)CSS
  • Bring dynamic content into your presentation
    • Because it's the web, you know ;-)

Default transition types

Webpresent includes different transitions:

  • Fade (default)
  • Fade in from the left
  • Fade in from the right
  • Fade in from the bottom
  • Fade in from the top
  • No transition at all
  • Your custom transition
    • All using CSS
